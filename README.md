## KDE's kimpanel implementation for GNOME Shell

### Changes and Improvements:
- Implemented system accent color support.
- Optimized the candidate panel to visually blend with native GNOME applications.
- Relocated the indicator to the rightmost edge of the status area.
- Resized the input method switcher popup.


### Install requirements  
```
gettext  
cmake
```

If you'd like to install it as user, run ./install.sh under the root of source.
